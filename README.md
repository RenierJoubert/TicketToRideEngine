# Ticket to Ride Optimized

A greedy pathfinding optimization engine for the board game *Ticket to Ride*, implemented in Java. This project applies graph theory and combinatorial optimization to compute maximum-value route selections under resource constraints. Namely, the Ticket to Ride Engine solves the constrained optimization problem in the board game *Ticket to Ride*, wherein given a set of destination tickets and a 45-train budget, determine which tickets to complete in order to maximize total score.
